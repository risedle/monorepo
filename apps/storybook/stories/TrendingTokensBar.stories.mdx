# Trending Tokens Bar

**TrendingTokensBar** is a stateless or presentational component.

-   **TrendingTokensBar** doesn't fetch [`TokenInfo`][tokeninfo] data directly.
-   `tokens` props are passed by parent container component:
    **TrendingTokensBarContainer**.
-   **TrendingTokensBar** takes props list of [`TokenInfo`][tokeninfo] and
    render DOM only.

For any amount of `tokens` the **TrendingTokensBar** will fill current
viewport.

Example usage:

```typescript
import { TrendingTokensBar } from "@risedle/ui";

export const Component = () => {
    // Fetch data or something
    return <TrendingTokensBar tokens={{ data }} />;
};
```
